<install Totalsegmatator前>
執行檔都被安裝在 ~/.local/bin 下，但這個目錄不在你的 PATH 環境變數裡，所以你在終端機直接輸入命令時可能找不到它們。直接用 shell 命令修改 .bashrc。
echo 'export PATH=$PATH:/home/帳號名/.local/bin' >> ~/.bashrc
→ 把路徑加到你的 .bashrc 裡（永久生效）。

source ~/.bashrc
→ 讓修改立即生效，不用重開終端機。

echo $PATH
確認 /home/sandy0317/.local/bin 已經在 PATH 裡
